# Nome do Projeto: myList 


## Resumo 
Projeto myList serve para que os usuários criem, visualizem, editem e marquem como concluídas as tarefas que precisam realizar. 
O principal objetivo é proporcionar uma forma simples e eficiente de gerenciar compromissos, prazos e prioridades. 

## Funcionalidades:
  * Adicionar tarefas;
  * Marcar tarefas como concluídas;
  * Editar tarefas;
  * Excluir tarefas.
